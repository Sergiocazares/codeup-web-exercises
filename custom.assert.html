================================= BUILD A TESTING FUNCTION Create a new html file in codeup-web-exercises called “custom-assert.html” with script tags to complete the exercise below: Group up! With your partners, figure out how to build a function that tests other functions. The function, called testFunction, should be defined with three parameters: 1. the name of the function to test (use a parameter name of callback) 2. an input to the function to test 3. the expected output of the test function when passed the input The function should return true or false based on whether or not the passed function returns the passed output based on the passed input. Test your testing function with the following flawed function which should return false: function returnInputAsOutput(input) { return “input”; } What should be fixed to make returnInputAsOutput work correctly? Using your function on the function above may look like this: console.log(testFunction(returnInputAsOutput, ‘bob’, ‘bob’)); console.log(testFunction(returnInputAsOutput, 2, 2)); console.log(testFunction(returnInputAsOutput, true, true));
